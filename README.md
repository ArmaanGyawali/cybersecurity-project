
# Phishing Email Detector

This is project for the B207 module. I built this program to detect if an email is phishing or safe using a Random Forest model.

## What it does
- It reads an email dataset.
- It uses machine learning to classify emails.
- It saves every result into a database automatically.

## How to use it
If you want to run this code, just follow these steps:

1. Clone this repository to your computer.
2. Run my setup file to get the libraries installed:
   ```bash
   bash setup_file.sh
3. to run it type python3 spam_detector.py in terminal
